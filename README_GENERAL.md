# Semana 11 - Artefacto del entorno: AgroData Integridad Pro

Tema: **Manejo de archivos y datos simples; integridad y organización de información**.

Este paquete contiene una actividad tipo **artefacto del entorno** para trabajar en VS Code. El objetivo es que el estudiante configure un pequeño sistema web, instale librerías, revise archivos de datos `.csv` y `.json`, complete validaciones, genere reportes y entregue un README breve con la explicación de su mini-solución.

## Estructura del paquete

```text
semana11-artefacto-entorno-agrodata-pro-v1/
├── carpeta_estudiante/

```

## Qué entregar a estudiantes

Entregar únicamente:

```text
carpeta_estudiante/
```

## Qué conservar el profesor

Conservar:

```text
carpeta_profesor/
```

La carpeta del profesor incluye la solución final, una guía detallada, respuestas esperadas y rúbrica de evaluación.
